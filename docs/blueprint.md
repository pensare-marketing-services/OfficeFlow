# **App Name**: OfficeFlow

## Core Features:

- User Authentication: Secure authentication system for admin and employees with role-based access control.
- Task Assignment: Admin can assign tasks to employees, specifying deadlines and priorities.
- Task Management: Employees can view assigned tasks, update their status, and add progress notes.
- Reporting Tool: Admin can create reports based on task progress, employee performance, and project timelines, and generate PDF reports for clients using AI-powered summaries.
- Progress Summarization: Use a tool to automatically summarize the progress on various projects, in a way that's client-facing and hides the underlying details.
- Notifications: Real-time notifications for task assignments, updates, and deadlines.

## Style Guidelines:

- Primary color: Dark, muted blue (#34495E) for a professional and trustworthy feel.
- Background color: Light gray (#F4F6F6), a desaturated variant of the primary, for a clean, spacious layout.
- Accent color: Soft orange (#E67E22), contrasting for important actions and highlights.
- Body font: 'PT Sans', a modern sans-serif for clear readability in reports and task descriptions.
- Headline font: 'Space Grotesk', a futuristic sans-serif for a tech-forward image in headings and titles.
- Consistent use of simple, professional icons from a set like Material Design Icons to represent task status, departments, and report types.
- Clean, card-based layout for tasks and reports, promoting ease of navigation and content scanning.
- Subtle transitions and loading animations to enhance user experience without being distracting.
